# ToDoアプリ

JavaScript学習のため、動画を見ながら作成したToDoアプリです。
ローカルストレージに保存できるので、ページを再読み込みしてもデータが残ります。

## 公開URL
👉 [デモページはこちら](https://miyu-code07.github.io/todolist-movie/)

## 主な機能
- タスクの追加
- タスクの削除
- ローカルストレージへの保存

## 使い方
1. タスクを入力して「+」ボタンを押す
2. タスクを終了したらチェックボタンを押す
3. 必要に応じてゴミ箱ボタンでタスクを消す

## 使用技術
- HTML
- CSS
- JavaScript
